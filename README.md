# cncguy-web

Static marketing site for the CNC Guy family of tools — CNCGuy Edit, CNCGuy Studio, and DNCGuy. Built with Astro. Deployed to Cloudflare Pages at cncguy.com.

## Local development

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static output to ./dist
npm run preview  # preview the production build locally
```

## Pages

- `/` — brand hub: hero, products grid, built-by-a-machinist, features, email capture
- `/edit` — CNCGuy Edit product page + download
- `/notify` — placeholder for Studio/DNCGuy notification signup

## Stack

- **Astro 6** (static)
- Raw CSS with the CNC Guy design system tokens (no Tailwind, no component framework)
- **Oswald**, **Source Sans 3**, **JetBrains Mono** — loaded via Google Fonts in `src/styles/tokens.css`
- No runtime JS beyond standard anchors and a stub subscribe form

## Design system

Tokens and component CSS were generated by Claude Design and preserved verbatim:

- `src/styles/tokens.css` — colors, typography, spacing, radii, motion
- `src/styles/site.css` — component classes (`cta-primary`, `feat-tile`, `cmp`, `chrome`, etc.)
- `src/components/` — Astro components (Nav, EditHero, FeatureTiles, Comparison, BetaHonesty, DownloadBlock, Footer)

## Deploy — Cloudflare Pages

1. Push this repo to GitHub.
2. In Cloudflare: Workers & Pages → Create → Connect to Git → pick this repo.
3. Build command: `npm run build`
4. Build output directory: `dist`
5. Deploy. First deploy takes ~60s.

Custom domain (`cncguy.com`):
- In the Cloudflare Pages project, add `cncguy.com` under Custom Domains.
- Cloudflare will display the DNS records to add at the registrar (123cheapdomains).

## Brand discipline (do not violate)

This site holds to a strict palette and typography. Summarized:

- Orange (`#d9601f`) appears on **one** element per page — the primary CTA. Exceptions on `/edit` only: the inline `[BETA]` chip, and a 2px orange left border on the "CNCGuy Edit Beta" column of the comparison table.
- No green/blue/red status colors on the marketing site (those are app-internal).
- Dark theme only for v1.0.
- Oswald for headings (uppercase), Source Sans 3 for body, JetBrains Mono for eyebrows and meta.
- Never use "revolutionize / seamless / game-changing / leverage / empower" in copy.
